CSS 의미

Cascading Style Sheets

스타일은 상위에서 하위로 상속된다.

왜 만들었을까?

정보(HTML)를 시각적으로 꾸미기 위해 고안되었다.

그럼 HTML에는 디자인 요소가 없었을까?

최소한의 디자인 요소는 존재했다.

Selector

선택하고 선언한다.

Selector 의 종류

* { color: red; }

전체 선택자

h1 { color: red; }

타입(or 태그) 선택자

.class { color: red; }

클래스 선택자

#id { color: red; }

아이디 선택자

ul li { color: red; }

하위 선택자

ul > li { color: red; }

자식 선택자

ul, ol { color: red; }

요소(Element, Element) 선택자

E + F { color: red; }

인접 형제 선택자

E ~ F { color: red; }

일반 형제 선택자

E[attr] { color: red; }

속성 선택자

E:link { color: red; }

링크 선택자

E:active { color: red; }

동적 선택자

E:nth-child(n) { color: red; }

가상 선택자

E:checked { color: red; }

상태 선택자

E:before { color: red; }

가상 엘리먼트 선택자

Selector 우선 순위

!important > style > id > class > type(or tag) > *

점수 매기기

Typo graphy

쓰여진 정보를 정확히 전달하는것

font-family: 'Nanum Gothic', sans-serif, serif

폰트를 지정한다.

font-size: 20px

폰트의 크기를 지정한다.

color: red

폰트의 컬러를 지정한다.

text-align: left

폰트를 정렬한다.

Layout

Inline Element

컨텐츠 영역을 사용한다.

Block-Level Element

부모 엘리먼트가 허용하는 가로 공간 전체를 사용한다.

Box Model

태그의 위치, 크기, 여백등을 지정한다.

{width: 100px; height: 100px;}

태그의 가로 / 세로 폭을 지정한다.

{border: #DDD 1px solid;}

태그의 외각선을 지정한다.

{margin: 0 0 0 0;}

태그와, 태그 사이의 여백을 지정한다.

{padding: 0 0 0 0;}

컨텐츠와, 외각선 사이의 여백을 지정한다.

{box-sizing: content-box;}

엘리먼트의 width, height 크기를 결정한다.

content-box

border-box

Positioning

{position: static;}

별도 위치가 지정되지 않는다.(초기값이 반영한다)

{position: relative;}

부모 엘리먼트를 기준으로 상대적 위치를 지정한다.

{position: absolute;}

부모 엘리먼트를 기준으로 절대적 위치를 지정한다.

{position: fixed;}

브라우저 화면을 기준으로 상대적 위치를 지정한다.

{float: left;}

엘리먼트를 원하는 위치에 정렬 시킨다.
